Robocon 2012

Project Description

The THEME for ROBOCON 2012 was “Peng On Dai Gat”, meaning peace and prosperity. A manual robot and two autonomous robots were to be fabricated with given weight and size constraints. Cylindrical "buns" were to be collected from variable heights upto 1.5 metres and were to be put into a basket brought by manual robot. Fabrication of PCBs was done using Eagle 5.10 for ARDUINO Mega2560 and power supply was done. Magnetometers (digital Compass I2C communication) were calibrated and tested to use for turning of autonomous robots.
