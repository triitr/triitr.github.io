Project Description

The project consisted of designing and fabricating a set of two autonomous robots and one manual robot which will act together to make a system of three pyramids of Egypt (Khufu, Khafraa and Mankaura). The manual robot was controlled using a PS3 (Playstation) Remote which was interfaced and customized to suit our requirements. The autonomous robots were controlled via the input from sensors and the programme code fed into them.
